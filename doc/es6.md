## EC6

#### 变量

- var  :
  - 可以重复声明
  - 没有快作用域
  - 不能限制修改
- let : 声明变量
- const : 声明常量 
  - 禁止重复声明
  - 控制修改
  - 支持块级作用域

#### 作用域

- 传统 : 函数级
- ES6 : 块级

#### 函数

- 箭头函数

  - ```
    function(参数){
    }
    
    //箭头函数
    (参数)=>{}
    ```

    - 如果1个参数，`()` 也可以省略
    - 如果1条语句return,`{}`也可以省略

- 参数展开

  - 剩余参数 : 必须是最后一个
  - 展开数组 : 直接把数组写入

#### 系统对象

- Array
  - Map : 映射
  - reduce : 减少
  - forEach ： 遍历
  - filter : 过滤
